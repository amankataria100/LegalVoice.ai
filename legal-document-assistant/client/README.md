# Legal Document Assistant Client

This project is a legal document assistant web application built with React. It provides users with a voice or chat interface for selecting legal documents, a form for entering user details, an OCR document upload feature, and explanations for supported documents. Additionally, users can view completed forms.

## Features

- **Home Page**: The landing page of the application.
- **Voice/Chat Interface**: An interactive interface for selecting legal documents using voice commands or chat.
- **User Details Form**: A form for users to enter their personal information.
- **OCR Document Upload**: A page for users to upload documents for Optical Character Recognition processing.
- **Supported Documents Explanation**: A page that provides explanations of the types of documents supported by the application.
- **Completed Forms Showcase**: A page that displays a list of completed forms for user reference.

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm (Node Package Manager)

### Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the client directory:
   ```
   cd legal-document-assistant/client
   ```

3. Install the dependencies:
   ```
   npm install
   ```

### Running the Application

To start the development server, run:
```
npm start
```

The application will be available at `http://localhost:3000`.

### Building for Production

To create a production build, run:
```
npm run build
```

This will generate a `build` directory with the optimized production files.
